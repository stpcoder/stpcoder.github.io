# Current Project Status

Last updated: 2026-03-23

## What this repo is right now

This repository currently serves the live `https://stpcoder.github.io/` site from the `main` branch root.

There are two layers in the repo:

- Source app: `liquid-glass/`
- Deployed static output: root `index.html` and root `assets/*`

The live site is built from `liquid-glass/` and then copied to the root for GitHub Pages deployment.

## Active runtime modes

The current site has three actual styles and one performance mode:

- `Liquid Glass`
- `Terminal`
- `macOS Desktop`
- `Reduced Graphics` toggle inside `Liquid Glass`

Important distinction:

- `Reduced Graphics` is not a separate style.
- It is a performance mode for `Liquid Glass`.

## Current app entry flow

- Default entry: `Liquid Glass`
- Style menu options: `Terminal`, `macOS Desktop`
- Reduced mode: automatic on lower-end devices, user-toggleable in the settings menu

Relevant files:

- `liquid-glass/src/App.jsx`
- `liquid-glass/src/contexts/StyleContext.jsx`
- `liquid-glass/src/components/StyleSwitcher.jsx`
- `liquid-glass/src/components/styles/LiquidGlassView.jsx`

## Data model in the current repo

There is no surviving dedicated historical `master-profile.json` from the earlier discussion.

The current data files that exist are:

- `data/resume-data.json`
- `liquid-glass/src/data/resume-data.json`

### Current problem

Those two files are not fully synchronized.

Known differences verified on 2026-03-23:

- `POSTECH Programming Contest` has different `featured` values between the two files.
- `Progressive Caching for Video Diffusion Model` appears in a different position between the two files.

### Current recommendation

Treat `data/resume-data.json` as the intended canonical site data until an explicit sync script exists.
Then sync `liquid-glass/src/data/resume-data.json` from it.

## Reduced Graphics mode

Reduced mode exists to keep the Liquid Glass experience usable on lower-end or Windows machines.

Current behavior:

- Auto-enabled on lower-end devices based on browser hints
- Can be manually toggled in the style switcher
- Keeps the glass look while reducing motion and rendering cost

Key implementation files:

- `liquid-glass/src/contexts/StyleContext.jsx`
- `liquid-glass/src/components/styles/LiquidGlassView.jsx`
- `liquid-glass/src/components/Scene.jsx`
- `liquid-glass/src/components/GlassBubble.jsx`
- `liquid-glass/src/App.css`

## Build and deploy flow

From this repo:

```bash
cd liquid-glass
npm install
npm run build
cd ..
cp liquid-glass/dist/index.html index.html
rsync -a liquid-glass/dist/assets/ assets/
git add index.html assets liquid-glass
```
